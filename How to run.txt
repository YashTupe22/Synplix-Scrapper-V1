Vercel-native setup (recommended):

1. Install Node.js 18+.
2. Run:
   npm install
3. Local dev (optional):
   vercel dev
4. Deploy to Vercel:
   vercel --prod

How it works:
- Frontend is served from /public
- API endpoint is /api/generate (Node.js + Google Places API)
- CSV is generated in memory and downloaded from browser (no filesystem dependency)

Important:
- max_results is limited to 100 per request for runtime safety.
- Set GOOGLE_PLACES_API_KEY in Vercel environment variables.
